<h1 align="center">Videl</h1>
 
<p align="center">  
  <strong>
    Videl
  </strong>
  
  is a component framework for Vue2. Based on CSS framework Bulma.
</p>

## Install

``` bash
# NPM
npm install videl --save-dev
```

## Use

```javascript
import Vue from 'vue'
import Videl from 'videl'

Vue.use(Videl)
```
